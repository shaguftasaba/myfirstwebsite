# Shagufta Saba — Premium AI Portfolio

A premium React portfolio built with Vite, Tailwind CSS, and Framer Motion for a B.Com student passionate about Artificial Intelligence.

## Overview

This project is designed for deployment on Vercel and includes:

- Clean, Apple-inspired UI
- Light and dark mode
- Responsive layout with modern motion
- Section navigation for Home, About, Projects, and Contact
- Tailwind CSS styling and reusable components

## Setup

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Push this repository to GitHub.
2. Create a new Vercel project and connect the GitHub repo.
3. Use `npm install` and `npm run build` as the build commands.
4. Set the output directory to `dist`.

## Folder Structure

- `src/`
  - `App.jsx` — main page composition
  - `main.jsx` — entry point
  - `index.css` — global Tailwind styling
  - `components/` — reusable components and sections
